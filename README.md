# Translation Tool
- Tool to translate videos to different language.
- Translates the text and dubs the voice


## Python Version 
Python 3.13.5

## Activate Virtual Env
```bash 
Source .venv/bin/activate
```

## Run the script

```python
# For Frontend
cd frontend && npm run dev
```

```python
# For backend
cd backend && python run.py
```
